## Get started in 3 steps

<Steps>
  <Step title="Install Motus">
    ```bash
    pip install lithosai-motus
    ```
  </Step>
  <Step title="Build your first agent">
    ```python
    from motus.agent import ReActAgent
    from motus.models import OpenAIChatClient
    from motus.tools import tool

    @tool
    async def greet(name: str) -> str:
        """Greet someone by name."""
        return f"Hello, {name}!"

    client = OpenAIChatClient(api_key="sk-...")
    agent = ReActAgent(client=client, model_name="gpt-4o", tools=[greet])
    ```
  </Step>
  <Step title="Serve or deploy">
    ```bash
    # Run locally
    motus serve start myapp:agent --port 8000

    # Deploy to the cloud
    motus deploy --name my-agent myapp:agent
    ```
  </Step>
</Steps>

## Supported frameworks

Motus works out of the box with agents built using:

- **Motus** native `ReActAgent`
- **OpenAI Agents SDK** — drop-in import replacement
- **Anthropic SDK** — `ToolRunner` adapter
- **Google ADK** — native `Agent` subclass
- **Plain Python** — any callable with the right signature
